Detailed Description

A node of the list of processes in fibered_processes_list, a fiber-enabled process.

A fibered_process is a process in which at least one thread has been converted to a fiber, so a fibered_process always has at least one fiber in the fibered_process::fibers_list field. Moreover the first element of the list has fibered_process::prev pointing to NULL and the last element of the list has fibered_process::next pointing to NULL.
